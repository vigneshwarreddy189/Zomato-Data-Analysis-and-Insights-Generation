# Zomato-Data-Analysis-and-Insights-Generation

 Zomato Data Analysis & Insights Generation
 ##Project Overview

This project focuses on analyzing Zomato restaurant data to uncover meaningful insights related to restaurant distribution, pricing, ratings, customer preferences, and online ordering behavior.
Using Python, Excel, SQL, and Power BI, the project transforms raw data into interactive dashboards and actionable insights useful for business decision-making.

## Objectives

1. Understand restaurant presence across locations and cuisines

2. Analyze the relationship between cost, ratings, and customer votes

3. Identify popular restaurant types and dishes

4. Study online ordering and table booking trends

5  Build interactive dashboards for stakeholder insights

## Tools & Technologies

1. Python (Pandas, NumPy, Matplotlib, Seaborn)

2. Excel (Initial data cleaning & preprocessing)

3. SQL (Query-based analysis)

4. Power BI (Interactive dashboards & visual storytelling)

## Dataset

Zomato restaurant dataset (cleaned using Excel & Python)

Includes:

1.Restaurant name & location

2. Cuisines & restaurant type

3. Approx cost for two

4. Ratings & votes

5. Online order & table booking availability
